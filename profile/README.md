## What We Build

Gogure Interactive is an independent collective of developers, designers, and systems thinkers building tools that put the user first — not the platform, not the algorithm, not the business model.

Our software is:

- **Free** — no paywalls, no freemium traps, no dark patterns
- **Private** — your data stays yours, architecture-first privacy
- **Personal** — built for real use cases, not enterprise dashboards

We operate at the intersection of applied AI, interface design, and systems engineering. Every project starts from first principles: *what does this tool actually need to do, and what is the minimal, correct way to do it?*

## Principles

```
01  Privacy is architecture, not a checkbox.
02  Complexity is earned, not defaulted to.
03  The best interface is the one you don't notice.
04  Open source where possible. Closed where necessary.
05  Ship. Iterate. Respect the user's time.
```
